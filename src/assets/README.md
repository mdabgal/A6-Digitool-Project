 A6-DigiTool-Project


 # A litte Description:

   * "This Project is a React-based Shopping cart application where users can add products, view the cart, and proccedd to checkout"


   # Technology use this project:

   * React.js
   * Tailwind Css
   * DaisUI
   * JavaScript (Es6+)
   * React-Toastify
   * Json Api
   * google font - (Manrope)


   # Features about my project:

   1. Dinamic Cart Count:
     * The number of products is displayed in the Navbar. Clicking the "proceed to Checkout" button reets the cart.

     2. Instant Feedback: 
     * Clicking the "Buy Now" button shows an “Added to cart” message directly on the button.

     3. Product & Cart Toggling: 
     * By default, the Products section is visible. Clicking the Cart button displays the Cart section.
